Para usar a biblioteca de forma correta é necessário o seguinte comando de compilação:

g++ nome_arquivo.cpp -o nome_arquivo grafos.cpp

Dessa forma você estará utilizando a implementação correta da biblioteca.

O código de exemplo para o uso da biblioteca se encontra no "main.cpp".
